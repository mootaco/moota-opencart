# moota-opencart

## Development
Run 

    ./link.sh <TARGET_DIR>
  
where `TARGET_DIR` is the root directory  
of your OpenCart installation.

e.g.:  

Opencart is in: `~/Sites/localhost/opencart`  
and there is this three folder in there: `admin`, `catalog`, `system`  

Run:

    ./link.sh ~/Sites/localhost/opencart
